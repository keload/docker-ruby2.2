# docker-ruby2.2

A docker image containing

- ruby 2.2.4p230
- Compass 1.0.3 (Polaris)
- git version 1.9.1
- nodeJS v0.10.25
- npm 1.3.10
- grunt-cli v1.2.0 / grunt v0.4.5
- Bower 1.7.9


## Getting Started

`docker run -i -t --rm=true keload/ruby2.2`

or

`docker pull keload/ruby2.2`
